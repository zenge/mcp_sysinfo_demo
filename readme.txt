================================================================================
  mcp_sysinfo_demo — 系统信息查询脚本
================================================================================

项目简介
--------
本项目包含一个用于查询主机名和 IP 地址的 Shell 脚本，适用于日常运维和
快速诊断场景。

================================================================================

脚本列表
--------

  1. get_host_info.sh     查询主机名、本地 IP 地址和公网 IP

================================================================================

get_host_info.sh 详细说明
--------------------------

功能:
  打印当前系统的主机名、所有活跃网络接口的 IPv4 地址，并尝试获取公网出口 IP。

适用环境:
  - Linux（GNU），推荐 Ubuntu / Debian / CentOS / RHEL / Fedora 等
  - macOS（BSD）
  - 需要 Bash 4.0+（脚本使用 `set -euo pipefail` 等语法）

依赖工具:
  - 必需: bash, hostname
  - 网络工具 (满足其一即可):
      Linux:    ip 命令 或 ifconfig
      macOS:    ifconfig (系统自带)
  - 可选: curl（用于获取公网 IP,没有 curl 时此部分静默跳过）

如何执行:
  # 1) 添加可执行权限
  chmod +x get_host_info.sh

  # 2) 直接运行
  ./get_host_info.sh

  # 也可以用 bash 显式调用
  bash get_host_info.sh

是否需要参数:
  不需要任何参数。脚本无命令行选项，直接运行即可。

输出示例 (macOS):
  === 主机名 ===
    macbook-pro.local

  === IP 地址 ===
    en0:      192.168.1.100
    utun0:    10.8.0.5

  === 公网 IP (可选) ===
    203.0.113.42

输出示例 (Linux):
  === 主机名 ===
    web-server-01

  === IP 地址 ===
    eth0:     10.0.0.5

  === 公网 IP (可选) ===
    203.0.113.42

备注:
  - 脚本会自动检测操作系统 (uname -s) 并选择正确的命令路径。
  - 回环地址 127.0.0.1 会被过滤，不会在输出中显示。
  - 公网 IP 查询依赖外部服务 (ifconfig.me)，超时设置为 3 秒，失败时静默
    跳过，不影响脚本的正常退出码。
  - 终端支持颜色时，输出会有彩色高亮；重定向到文件时颜色自动禁用。

================================================================================

常见问题
--------

Q: 运行时报 "Permission denied"
A: 需要先执行 `chmod +x get_host_info.sh` 赋予可执行权限。

Q: Linux 下显示 "无可用网络工具"
A: 安装 iproute2 (`apt install iproute2` 或 `yum install iproute`) 或
   net-tools (`apt install net-tools`)。

Q: macOS 下 ifconfig 报 command not found
A: 检查 PATH 是否包含 /sbin (export PATH="$PATH:/sbin")。
